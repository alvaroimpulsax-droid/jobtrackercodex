# JobTracker URL Bridge

## Chrome / Edge
1. Abre `chrome://extensions`.
2. Activa "Developer mode".
3. Click en "Load unpacked" y selecciona `apps/extension`.

## Firefox
1. Abre `about:debugging`.
2. Click en "This Firefox".
3. "Load Temporary Add-on" y selecciona `apps/extension/manifest.json`.

La extensión enviará la URL activa al agente local si está en ejecución.
